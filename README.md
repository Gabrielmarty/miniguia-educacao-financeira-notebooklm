# 📘 Caderno Temático – Educação Financeira com NotebookLM

## 📌 Contexto e Objetivos

Este projeto foi desenvolvido como parte de um desafio da plataforma **DIO** com o objetivo de explorar o uso da ferramenta NotebookLM como apoio ao aprendizado.

O tema escolhido foi **Educação Financeira Básica**, buscando compreender conceitos essenciais sobre organização do dinheiro, planejamento financeiro e investimentos iniciais.

Objetivos do estudo:

* Compreender os princípios da educação financeira
* Aprender a organizar orçamento pessoal
* Entender conceitos básicos de investimento
* Utilizar IA como ferramenta de apoio ao estudo

# 🤖 Engenharia de Prompts

Durante o estudo foram realizados testes de perguntas para extrair informações relevantes.

### Prompt 1

**Pergunta:**
O que é educação financeira?

**Resposta resumida:**
Educação financeira é o conhecimento que permite administrar melhor o dinheiro, controlar gastos, planejar o futuro e tomar decisões financeiras mais conscientes.

---

### Prompt 2

**Pergunta:**
Qual a importância de ter uma reserva de emergência?

**Resposta resumida:**
A reserva de emergência é um valor guardado para situações inesperadas, como perda de renda ou despesas médicas.

---

### Prompt 3

**Pergunta:**
Qual a diferença entre poupar e investir?

**Resposta resumida:**
Poupar significa guardar dinheiro para uso futuro. Investir significa aplicar o dinheiro com objetivo de gerar rendimento.

# 📖 Miniguia de Estudo

## 💰 Educação Financeira

Educação financeira é a habilidade de administrar recursos financeiros de forma consciente, equilibrando gastos, economias e investimentos.

Ela ajuda a evitar dívidas e a alcançar objetivos financeiros de curto, médio e longo prazo.

---

## 📊 Planejamento Financeiro

O planejamento financeiro envolve organizar receitas e despesas para atingir objetivos financeiros.

Passos importantes:

* controlar gastos
* definir metas financeiras
* criar reserva de emergência
* investir com planejamento

---

# 📚 Glossário

**Orçamento:** controle das receitas e despesas.

**Reserva de emergência:** valor guardado para imprevistos.

**Investimento:** aplicação de dinheiro com expectativa de rendimento.

**Liquidez:** facilidade de transformar um investimento em dinheiro.
